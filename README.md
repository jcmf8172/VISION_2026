# VISION_2026

Proyecto de programación iniciado en **FreeBASIC 1.10.1**.

## Requisitos
- Compilador FreeBASIC (fbc) versión 1.10.1 o superior.

## Cómo compilar y ejecutar
Para compilar el proyecto, usa el siguiente comando:
```bash
fbc main.bas
```

Para ejecutar el programa:
```bash
./main
```

## Estructura del Proyecto
- `main.bas`: Punto de entrada principal del programa.
